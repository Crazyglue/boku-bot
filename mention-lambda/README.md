# Mention Lambda

Used to handle any time boku is mentioned, for example `@boku show me a dank meme`

## Required env vars

* SLACK_ACCESS_TOKEN
* SLACK_VERIFICATION_TOKEN
* IMAGE_FLIP_USERNAME
* IMAGE_FLIP_PASSWORD
* DYNAMO_TABLE_NAME
* BOKU_MEME_FEEDBACK_TABLE
* GOOGLE_API_KEY
* GOOGLE_SEARCH_ENGINE_ID
